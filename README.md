# TaskFlow (Ethara.AI assignment)

**TaskFlow** is a minimalist, high-performance project management tool built with the MERN stack. Designed with a high-visibility, developer-centric interface, it allows teams to manage projects and tasks with speed and clarity.

## Features

- **One-Step Project Initialization:** Admins can add team members during project creation to get started immediately.
- **Role-Based Access Control:** Distinct views and permissions for **Admins** and **Members**.
- **Interactive Task Management:** Create, assign, and track tasks with a modern, high-contrast UI.
- **High-Visibility UI:** Optimized for dark mode with sharp contrast for maximum readability.
- **Seamless Navigation:** A sticky, glassmorphism-style navbar for easy access to dashboard and profile settings.

## Tech Stack

- **Frontend:** [React](https://reactjs.org/), [Tailwind CSS](https://tailwindcss.com/), [Lucide React](https://lucide.dev/) (Icons)
- **Backend:** [Node.js](https://nodejs.org/), [Express.js](https://expressjs.com/), [MongoDB](https://www.mongodb.com/) (Mongoose)
- **Authentication:** [JWT](https://jwt.io/) (JSON Web Tokens)[cite: 1]
- **API Client:** [Axios](https://axios-http.com/)[cite: 1]
